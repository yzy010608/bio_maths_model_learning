# 案例说明
## 数据说明
gene_expression_data.xlsx 文件中共有150个样本，其中A、B、C三种细胞各50个。

Gene_1和 Gene_2在 Cell_Line_A中高表达，在 B和 C中低表达。

Gene_3在 Cell_Line_B中特异性高表达。

Gene_4和 Gene_5在所有细胞系中表达水平相当（作为背景噪声）

Cell_Line 列是细胞标签列

## 分析要求
请根据绘制出三个样本的PCA得分图，判断不同种样本之间设置是否合理，同种样本之间实验是否有误
